# Tajweed Quiz Teacher MVP
The foundational infrastructure and Phase 1-4 are actively complete.

## Next Steps
1. Start the development server using `npm run dev`.
2. Register an account and test the Teacher role.
3. Make sure to get an API key from [OpenRouter](https://openrouter.ai/) and add it to your `.env.local` as `OPENROUTER_API_KEY=your_key_here`.
4. Proceed to Phase 5 (Student Experience) to allow guest testing!

> Note: For any UI styling tweaks or missing shadcn components, you can add them easily utilizing `npx shadcn@latest add [component]`!
